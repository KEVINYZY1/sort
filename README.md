# sort
#####前言：
不同排序算法的实现，以及各公司的在线编程题。


######排序分为两大类：
内排序和外排序。  
######内排序：
所有排序过程中的记录都在内存中进行。  
######外排序：
在排序的过程中需要外存辅助存储记录过程。  

######内排的分类：
1.插入排序：直接插入排序，二分插入排序， 希尔排序  
2.选择排序：简单选择排序，堆排序  
3.交换排序：冒泡排序，快速排序  
4.归并排序  
5.基数排序
